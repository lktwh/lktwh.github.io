# 数组

# 定义数组

# 二维数组

定义

# 取地址运算：&

  • &：获取变量的地址，操作数必须是一个变量

  • 地址的大小是否与 int 相同取决于编译器

# 指针

  • 指针变量就是保存地址的变量
    • int *p = &i
  • 变量的值是内存的地址
    • 普通变量的值是实际的值
    • 指针变量的值是具有实际值的变量的地址

# 作为参数的指针

# 访问那个地址上的变量

  • * 是一个单目运算符，用来访问指针的值所表示的地址上的变量

  • 可以作左值，也可以作右值

# 指针应用

  • 函数返回多个值，某些值只能通过指针返回

# 指针与数组

传入函数的数组成了什么？

  • 函数参数表中的数组实际上是指针
    • sizeof(a) == sizeof(int *)
    • 但是可以使用数组的运算符 [] 进行运算
# 数组变量是特殊的指针
  • 数组变量本身表达地址，因此
    • int a [10]; int *p = a; // 不需要 &
    • 但是数组的单元表达的是变量 需要使用 & 取地址
    • a == &a[0]
  • [] 运算符可以对数组做，也可以对指针做
  • p[0] <==> a[0]
  • * 运算符可以对指针做，也可以对数组做
    • *a <==> a[0]
  • 数组变量是 const 指针，所以不能被赋值，即
    • int a[] <==> int * const a
    • 因此，int b []; b = a 是错误的
    • 但是可以写成 int *q = a;

# 指针与 const

# 指针是 const（指针常量）
  • 表示一旦得到某个变量的地址，就不能再指向其他的变量
# 所指的是 const（常量指针）
  • 表示不能通过指针去修改那个变量（指针的指向可以修改，但是指针指向的值不能修改）
# 保护数组的值
  • 因为把数组传入函数时传递的是地址，所以函数内部可以修改数组的值
  • 为了保护数组的值不被函数破坏，可以将参数设置为 const
  • int sum(const int a[], int length);

# 空指针与野指针

# 空指针
  • 空指针用于给指针变量进行初始化： int *p = NULL
  • 空指针是不能进行访问的，即 *p = 100 是不允许的
